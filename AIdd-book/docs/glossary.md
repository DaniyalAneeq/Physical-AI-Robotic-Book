# Glossary

This glossary defines common terms used throughout the textbook to ensure consistency.

- **ROS (Robot Operating System):** A set of software libraries and tools that help you build robot applications.
- **Node:** A process that performs computation.
- **Topic:** A named bus over which nodes exchange messages.
- **Message:** A ROS data type, used when subscribing or publishing to a topic.
- **Service:** A request/response style of communication between nodes.
- **Action:** A long-running, feedback-providing version of a service.
- **URDF (Unified Robot Description Format):** An XML format for representing a robot model.
